How to play

Fill every row, every column, and every 3×3 box with the digits 1–9, each appearing exactly once.

Controls

  • Click a cell to select it, or use arrow keys to move the selection.
  • Type 1–9 to place a number in the selected cell.
  • 0 / Backspace / Delete: clear the cell (clears pencil marks if the cell is already empty).
  • Ctrl+Z or the ↶ button: revert the last action.

Pencil marks

Click Pencil to toggle pencil mode, or hold Shift / Ctrl / Alt while entering a digit.

Number-done indicator

When all 9 instances of a digit are placed, its pad button shows ✓ and can no longer be used as input.

Hotkeys

  • 1–9 — place digit in selected cell
  • Shift/Ctrl/Alt + 1–9 — toggle pencil mark
  • 0 / Backspace / Delete — clear cell or pencil marks
  • Ctrl+Z — undo
  • Arrow keys — move selection
  • p — pause / resume the timer
  • ssssss — press s six times in a row to reveal the solution (debug)
  • Esc — close any floating window

Difficulty variants

  • Easy → Expert: standard carving-based generation targeting a clue count for each level.
  • Extreme: try to produce a real 17-clue puzzle for up to 42 seconds. If the budget runs out without hitting 17, one is picked from a bundled verified 17-clue pool.
  • Extreme - Gen: keep carving indefinitely until the generator reaches a 17-clue puzzle.
  • Extreme - Pool: skip generation and pick a puzzle straight from the bundled 17-clue catalog.

Difficulty metric

The number after each puzzle ("N backtracks") comes from running a deterministic brute-force solver and counting how many times it had to backtrack. More backtracks = harder puzzle.
